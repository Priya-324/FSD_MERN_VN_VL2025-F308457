# College Info Portal (Frontend Only)

A responsive React app with five pages, dynamic routing, reusable components, and local JSON for department data.

## Features
- Home: Hero, “Why this college?”, Top departments, quick links
- About: History, Mission & Vision, Achievements
- Departments: All departments as cards with “View More”
- Department Details: Dynamic route `/departments/:slug`, courses, faculty, labs
- Contact: Working form, displays submitted details on same page

## Tech
- React + Vite
- react-router-dom
- Local JSON data (`src/data/departments.json`)

## Run locally
1. npm install
2. npm run dev
3. Open the local URL from the terminal

No backend required.
